# 🗑 ゴミ出しアラーム

シンプルなデザインのゴミ収集日アラームアプリです。好きな画像をアイコンに使えます。

## 機能

| 機能 | 内容 |
|------|------|
| 収集日登録 | ゴミの種類・曜日・隔週設定ができます |
| アラーム通知 | 何日前の何時に通知するかを設定できます |
| アイコン | 絵文字または好きな画像をアップロードできます |
| 初期セット | 可燃・不燃・資源化物・プラスチック・小型家電・粗大ごみが最初から入っています |
